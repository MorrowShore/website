{% filter commonmark_full %} ## Get Help, Give Suggestions, Report Bugs
 Discord  Web Chat  GitHub
Joining the **Drawpile Discord Server** is probably the easiest way to get help, since there's a number of nice people at the ready to answer questions and provide assistance. It's also a good place to keep up with the latest development, provide suggestions, report issues and so on. You can also jump directly into Drawpile's IRC chatroom using **the web chat**. Just press the "start" button and talk away, you don't need an account. If you already have an IRC client, you can join the *#drawpile* channel on libera.chat. You can also report stuff in the **GitHub Issue Tracker** directly if you like.
