Name Check Bot

Short answer: don't worry, you probably just didn't mark your session not suitable for minors (NSFM). You can do that by ticking the checkbox either when hosting the session or in a running session by clicking on Session at the top and then going into Settings.

The name check bot will automatically detect unacceptable or misleading session titles, as well as usernames in public sessions. Session names show up in public, even for personal sessions with a password. Drawpile gets used in schools and such, so the check is pretty strict if you don't mark your session NSFM.

The name check bot will never ban you and doesn't analyze what you're drawing. It only takes care of public names so that the human server admins don't need to. It sometimes gets stuff wrong, sorry about that, we try to fix it when it happens.

We know it's possible to circumvent the name check bot if you try hard enough. Please don't do that, even if you disagree with what the bot considers unacceptable. A human server admin can see that you're intentionally breaking the rules and will ban you for it.

When you use the public Drawpile server, you have to follow its rules. You can read them here. If you disagree with the rules on the public server, you can set up your own server instead and make your own rules. If you need help doing this, check out the help page!
