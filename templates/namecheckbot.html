Name Check Bot

When you use the public Drawpile server, you have to follow its rules. You can read them here.

The name check bot will automatically detect unacceptable or misleading session titles and usernames in open sessions. Session names show up in public, even if the session itself is private with a password, so the names have to adhere to the rules.

To avoid most cases of being bothered by the bot, properly mark your session as not suitable for minors (NSFM). You can do that by ticking the checkbox either when hosting the session or in a running session by clicking on Session at the top and then going into Settings.

The name check bot will never ban you and doesn't analyze what you're drawing. It only takes care of public names so that the human server admins don't need to.

We know it's possible to circumvent the name check bot if you doctor names enough. Don't do that, even if you disagree with what the bot considers unacceptable. A human server admin can see that you're intentionally breaking the rules and will ban you for it.

If you disagree with the rules on the public server, you can set up your own server instead and make your own rules. If you need help doing this, check out the help page!
