Community Ban Analyzer

To use this, join the offending session and go into Session → Event Log. Filter for "address". Alternatively, look in the server logs themselves.

Copy the results into the text field below and the table will take it apart. It will ignore extra data, so just copy a bunch of stuff and it will figure it out.

Include the Username into the "Description" field, along with the reason for the ban.

IP Address goes into the "Start of IP Range" field. Leave the "End of IP Range (inclusive)" field empty.

System Identifier and User ID go into those respective fields.

Make sure to read the instructions on bans. Also, remember to paste the data into the #community-moderators chat on Discord, so that others can check what you did, it helps avoid mistakes.
